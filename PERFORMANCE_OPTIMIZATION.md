# 性能优化计划

## 现状

Electron 应用体积大（~150MB）是框架本身的限制，无法根本解决。以下优化聚焦于**启动速度**和**运行性能**。

## 优化清单

### 1. 打包优化 ✅ 已完成
- [x] 排除不必要文件（docs、md、test）
- [x] 启用代码分割
- [x] 生产环境移除 console

### 2. 启动优化
- [ ] 延迟非关键工具检测（启动时只检测常用工具）
- [x] 组件懒加载

### 3. 运行时优化
- [x] 虚拟化长列表（包列表、工具列表）— 已有分页，无需额外虚拟化
- [ ] 优化组件重渲染（React.memo）

### 4. 资源优化
- [x] 压缩 docs 文件夹中的大图片（16MB → 0.6MB，减少 96%）

## 预期效果

- 启动时间：减少 30-50%
- 列表滚动：更流畅
- 内存占用：减少 10-20%
